About This Project

Tiffin Tonic — Healthy Tiffin Service System

This project is a web-based application developed as part of our college project submission for the 6th Semester, B.Tech (CSE) at NIIS Institute of Engineering and Technology, Chhatabar, Bhubaneswar.

The system is designed to solve a real problem: many students and working professionals depend on unhealthy fast food because they don't have time to cook. Our platform lets users browse healthy meals and place orders easily from any device.

💻 Technologies: HTML, CSS, JavaScript
